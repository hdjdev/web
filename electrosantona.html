Sitio web corporativo para Electro Santoña

VISITAR SITIO WEB MI PORTFOLIO

INTRODUCCIÓN

Durante mis prácticas profesionales en Tecnomedia Digital desarrollé el sitio web corporativo de Electro Santoña, una empresa dedicada a la reparación y venta de electrodomésticos. El objetivo fue crear una web clara, funcional y fácil de gestionar que mejorara su presencia digital y facilitara el contacto con clientes.

EL RETO

La empresa necesitaba un sitio web profesional que mostrara sus servicios, marcas y datos de contacto de forma clara, permitiendo además una gestión sencilla de los contenidos. El reto fue desarrollar una web accesible, responsive y orientada a un negocio local.

FUNCIONALIDADES PRINCIPALES

Claridad

Presentación clara de servicios y especialidades

Marcas

Sección de marcas con las que trabaja la empresa

Contacto

Información de contacto, horario y localización

Formulario

Formulario de contacto funcional

Responsive

Diseño responsive adaptado a dispositivos móviles

WordPress

Gestión de contenidos mediante WordPress

ARQUITECTURA Y FUNCIONAMIENTO

El proyecto se desarrolló utilizando WordPress como gestor de contenidos, permitiendo una estructura clara de páginas y una administración sencilla por parte del cliente. La web está organizada para facilitar la navegación y el acceso rápido a la información más relevante.

GESTIÓN DE CONTENIDOS

El uso de WordPress permite actualizar textos, servicios y contenidos de forma autónoma sin necesidad de conocimientos técnicos. La estructura del sitio está pensada para facilitar futuras ampliaciones y mejoras.

Diseño UI/UX con Figma

VER WIREFRAMES MI PORTFOLIO

INTRODUCCIÓN

Proyecto de diseño UI/UX enfocado en la creación de una interfaz clara e intuitiva para una aplicación web. El trabajo abarca desde la definición de la estructura y navegación hasta la creación de prototipos en Figma, priorizando la usabilidad y la claridad visual.

EL RETO

El objetivo del proyecto fue diseñar un prototipo de baja fidelidad para una aplicación web compuesta por tres pantallas clave, definiendo una estructura lógica y una navegación sencilla que permitiera al usuario interactuar de forma intuitiva con la información.

PROCESO DE DISEÑO

El proyecto se desarrolló siguiendo un proceso estructurado de diseño:

  • Análisis de requisitos y definición de necesidades
  • Boceteado inicial mediante wireframes
  • Diseño de pantallas y componentes en Figma
  • Revisión y ajustes finales del prototipo

ARQUITECTURA DE LA INTERFAZ

La estructura de la interfaz se diseñó teniendo en cuenta la jerarquía visual, la consistencia entre pantallas y la facilidad de navegación. Se emplearon grids y principios de diseño responsive para asegurar una base sólida y escalable.
